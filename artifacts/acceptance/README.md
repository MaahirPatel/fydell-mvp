# Acceptance evidence artifacts (V2)

Generated by `npm run test:acceptance-proofs`.

| File | Meaning |
|------|---------|
| `../acceptance-manifest.json` | Machine-readable gate statuses (four-state protocol) |
| `SUMMARY.json` | S0/S1 traceability score \(T_c\) |
| `SEC-*.json` … | Per-gate expected/actual proof payloads |

**Four-state statuses only:** `NOT_STARTED` | `IMPLEMENTED_UNVERIFIED` | `VERIFIED` | `BLOCKED`.

Do not mark a gate `VERIFIED` without a corresponding artifact file and passing runner output.
